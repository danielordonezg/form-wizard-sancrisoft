# Company Wizard

Un form wizard de 3 pasos para enviar información de empresa vía `/api/submit`.

## Tech stack
- Next.js 14 (React 18 + TypeScript)  
- styled-components  
- Zod

## Setup

```bash
git clone <repo-url>
cd company-wizard
npm install
npm run dev

Build & Run

npm run build
npm start

Test

npm test
